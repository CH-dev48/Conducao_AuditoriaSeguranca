# Auditoria de Segurança - Botium Toys

## Descrição
Projeto desenvolvido como atividade prática de auditoria interna de segurança da informação utilizando o framework NIST Cybersecurity Framework (NIST CSF).

## Objetivos
- Avaliar controles de segurança existentes
- Verificar conformidade com PCI DSS, GDPR e SOC
- Identificar riscos e vulnerabilidades
- Propor recomendações para mitigação de riscos

## Principais Resultados

### Controles Ausentes
- Least Privilege
- Separation of Duties
- IDS (Intrusion Detection System)
- Backups
- Encryption
- Password Management System
- Disaster Recovery Plan

### Controles Implementados
- Firewall
- Antivirus
- CCTV
- Fechaduras físicas
- Sistema de detecção de incêndio

## Conformidades Avaliadas
- PCI DSS
- GDPR
- SOC 1 / SOC 2

## Recomendações
1. Implementar criptografia dos dados.
2. Criar plano de recuperação de desastres.
3. Implementar backups automáticos.
4. Restringir acessos utilizando Least Privilege.
5. Implantar IDS para monitoramento de ameaças.
